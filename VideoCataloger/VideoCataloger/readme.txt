CatalogerSampleScripts.sln is a visual studio solutiuon that contains all 
Fast video cataloger samples. 

The solution allow you to get intellisense in visual studio as when 
working on scripts for Fast video cataloger. 

You can also compile the scripts to ensure they are valid c#.

During script development, add your own script to the solution.

To run the scripts you need to either load them from the consol window
or call from an action in fast video cataloger.
